Get Started

Add Zumly to your project in seconds.

NPM

npm install zumly

CDN

<link rel="stylesheet" href="https://unpkg.com/zumly/dist/zumly.css">
<script src="https://unpkg.com/zumly/dist/zumly.js"></script>

ES Module

import Zumly from 'zumly'
import 'zumly/css'

Hello World

const app = new Zumly({
  mount: '.canvas',
  initialView: 'home',
  views: { home, detail }
})
await app.init()
