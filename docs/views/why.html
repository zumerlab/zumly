Why Zumly

Add zoom navigation to any project with just HTML.

✎

Just HTML

Add zoom-me and data-to to any element and it becomes zoomable. No special syntax, no framework required. If you can write HTML, you can use Zumly.

▩

Independent Views

Unlike Prezi or canvas-based tools, views are loaded on demand from separate files, URLs, or components. You don't need everything on one giant canvas.

❖

Infinite Zoom Depth

Zoom into views that zoom into views, with no limit. Each level loads its own content on demand, so your interface can go as deep as you need.

⚙

Pluggable Drivers

CSS, WAAPI, GSAP, Anime.js, Motion, or your own. Swap animation engines without touching app code.

⚡

Smart Prefetching

Eager preload, hover prefetch, scan prefetch. Views are ready before the user clicks.

⚪

Zero Dependencies

Lightweight core with no deps. Works in any project: vanilla, React, Vue, Svelte, Angular. Animation libraries are optional.
